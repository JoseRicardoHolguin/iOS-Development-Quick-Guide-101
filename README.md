# iOS Development Quick Guide 101

Final project for Mobile Programming II course - A comprehensive reference guide for iOS development with Swift and SwiftUI.

## 📚 Topics Covered

1. Swift Basics
2. OOP in Swift
3. SwiftUI Fundamentals
4. MVVM + @Observable
5. NavigationStack & TabView
6. Lists & Tables
7. Animations & Transitions
8. REST APIs & JSON
9. MapKit
10. Camera & QR Reader

## 🌐 Live Demo

This guide is published on GitHub Pages at: https://josericardoholguin.github.io/iOS-Development-Quick-Guide-101/

## 🚀 Local Development

Simply open `index.html` in your browser to view the guide locally.
